Installation of gitlib
======================

Autoloading
:::::::::::

gitlib relies on class autoloading. It does not require any additional setup.

Using composer
::::::::::::::

Edit your ``composer.json`` file and add ``gitonomy/gitlib`` in section ``require``:

.. code-block:: json

    {
        "require": {
            "gitonomy/gitlib": "~0"
        }
    }
