Installation of gitlib
======================

Autoloading
-----------

gitlib relies on class autoloading. It does not require any additional
setup.

Using composer
--------------

Edit your `composer.json` file and add `gitonomy/gitlib` in section
`require`:

``` {.sourceCode .json}
{
    "require": {
        "gitonomy/gitlib": "^1.0"
    }
}
```
